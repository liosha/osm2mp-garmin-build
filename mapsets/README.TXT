Описание ключей, используемых в mapset-ах

# Common settings
(общие настройки)

prefix			префикс для имени файла сборки (архива)
countrycode		двухбуквенный код страны по ISO, нужен для адресного поиска
countryname		название страны
filename		имя файла сборки (архива)
fid			идентификатор карты, Garmin Family ID ( 1..9999 )
config			файл настроек для osm2mp
config_brokenmpoly	файл настроек для osm2mp (сборка поломанных полигонов, см fixmultipoly)
typ			путь к typ-файлу
readme			путь к файлу (или нескольким) readme, просто копируется в сборку
keys			дополнительные ключи osm2mp
url_base		адрес и путь к файлам с данными OSM (дампам), см srcalias
config_file_ftp		настройки FTP для загрузки результата
continue_mode		после перезапуска продолжить с региона, на котором прервались в прошлый раз ( 0*|1 )
make_house_search	формировать слой адресного поиска ( 0|1* )
mp_threads_num		количество параллельных потоков формирования MP-файлов
update_cfg		обновлять конфигурацию (SVN) перед началом процесса ( 0|1* )
skip_dl_src		не загружать дампы OSM ( 0*|1 )
skip_dl_bounds		не загружать полигоны границ ( 0*|1 )
skip_img_build		не формировать IMG ( 0*|1 )
skip_mp_upload		не загружать MP на сервер ( 0*|1 )

# Regions
(отдельные регионы)

name			имя карты, отображается навигатором
code			число, прибавляемое к fid из общих настроек для получения fid региона
alias			алиас региона, используется для имён мапсета, дампа (если нет srcalias), границы (если нет bound)
srcalias		имя дампа OSM, конечный URL будет таким: <url_base>/<srcalias|alias>.osm.<format>
srcurl			полный путь к дампу OSM (см srcalias)
bound			ID отношения (или алиас) границы, передаваемый getbound для загрузки
keys			дополнительные ключи osm2mp
skip_mapset		не формировать сборку, только скомпилировать IMG для использования в других мапсетах ( 0*|1 )
pre_clip		предварительно обрезать дамп по границе ( 0*|1 )
onering			объединить все полигоны границы одной линией ( yes|no* )
			такое нужно, если карта, например, состоит из нескольких островов
clipbound		объединить примыкающие полигоны границ в один, union ( yes|no* )
			например, чтобы получить единый полигон страны имея только полигоны регионов
skip_mp_upload		не загружать MP на сервер ( 0*|1 )
fixmultipoly		отдельно загрузить и скомпилировать полигоны, которые присутствуют в дампе не полностью ( yes|no* )
format			формат дампа OSM ( bz2|pbf* )

# Mapsets
(мапсеты, собираются из готовых регионов путем создания общего индекса)

name			имя карты, отображается навигатором
fid			идентификатор карты, Garmin Family ID ( 1..9999 )
filename		имя файла сборки (архива)
make_house_search	формировать слой адресного поиска ( 0|1* )
skip_mp_upload		не загружать MP на сервер ( 0*|1 )
parts			массив алиасов регионов, из которых формируется мапсет
			для каждого алиаса должно быть описание в секции Regions
