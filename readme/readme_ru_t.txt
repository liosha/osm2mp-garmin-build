ОПИСАНИЕ

	Карты для навигаторов GARMIN. Данные взяты из проекта OpenStreetMap. 
	Для конвертации использовались программы:
		osm2mp (http://code.google.com/p/osm2mp/)
		cgpsmapper 

ТРЕБОВАНИЯ
	
	Для установки в навигатор необходима программа MapSource или 
	BaseCamp. Для Linux можно использовать QLandkarteGT, но результат 
	может отличаться.

УСТАНОВКА В MAPSOURCE

	- Распаковывать архив в отдельную папку
	- Установить карту
		-Базовый вариант: запустить скрипт INSTALL.BAT от имени администратора
		-GMAPI: копировать папку FAMILY_XXX.gmapi\FAMILY_XXX.gmap (или ярлык на нее) в 
			папку %AppData%\Garmin\Maps
	- Подключить навигатор
	- Открыть программу MapSource
	- Выбрать установленную карту в списке карт (в левом верхнем углу на 
	  панели инструментов)
	- Включить инструмент "Выделение карт" на панели инструментов
	- Выделить карту (именно выделить/обвести мышкой, а не просто щелкнуть)
	- Нажать кнопку "Отправить на устройство" на панели инструментов

УСТАНОВКА В BASECAMP

	- Распаковывать архив в отдельную папку
	- Установить карту
		-Базовый вариант: запустить скрипт INSTALL.BAT от имени администратора
		-GMAPI: копировать папку FAMILY_XXX.gmapi\FAMILY_XXX.gmap (или ярлык на нее) в 
			папку %AppData%\Garmin\Maps
	- Подключить навигатор
	- Открыть программу BaseCamp
	- Выбрать карту в меню Карты 
	- Меню Карты -> Установить карты
	- Выбрать устройство
	- Продолжить
	- Выделить карту (именно выделить/обвести мышкой, а не просто щелкнуть)
	- Продолжить
	- Установить

УСТАНОВКА НА OSX

GMAPI
	- Распаковывать архив в отдельную папку
	- Открыть FAMILY_XXX.gmap
	- см УСТАНОВКА BASECAMP

Базовый вариант
	- Установить карту на ПК с Windows
	- Преобразовать карту с помощью MapConverter от Garmin
	  http://www8.garmin.com/support/download_details.jsp?id=3897
	- Загрузить на ПК с OSX программой Map Manager
	  http://www8.garmin.com/osx/
	- см УСТАНОВКА BASECAMP
